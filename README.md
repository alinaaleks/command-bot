# 📌 Command Bot

### 🇷🇺 Описание

Telegram-бот для быстрого доступа к командам и горячим клавишам (Git, CLI, Python, Conda, веб-разработка).

### Проблема

Частый поиск одних и тех же команд замедляет работу — понимаешь, что нужно, но не помнишь точную формулировку.

### Решение

Лёгкий справочник внутри Telegram с быстрым доступом к структурированным командам.

### Функционал

- Навигация по иерархии команд
- Быстрый доступ через Telegram
- Простое редактирование и масштабирование

### Технологии

- Telegram Bot API
- Telebothost (TBL scripting)
- Git и GitHub
- _(Первая версия: Python + Railway)_

### Как работает

1. `/start` → список программ
2. Выбор программы
3. Выбор категории
4. Получение списка команд

Структура:
**Программа → Категория → Команды**

### Результат

- Ускорение работы
- Снижение времени поиска
- Удобство в повседневном использовании

### 🇬🇧 Description

Telegram bot for quickly looking up commands and keyboard shortcuts for tools like Git, CLI, Python, Conda, and web development.

### Problem

Repeatedly searching for the same commands slows down workflow. You often know _what_ you need, but not the exact syntax.

### Solution

A lightweight Telegram-based reference system that provides instant access to structured command lists.

### Function

- Browse commands via hierarchical structure
- Fast access directly inside Telegram
- Easy to update and extend

### Tech Stack

- Telegram Bot API
- Telebothost (TBL scripting)
- Git & GitHub
- _(Initial version: Python + Railway)_

### How It Works

1. `/start` → list of tools (Git, Python, etc.)
2. Select tool
3. Select category
4. Get command list

Structure:
**Tool → Category → Commands**

### Result

- Faster workflow
- Reduced time spent searching
- Simple and practical daily usage
